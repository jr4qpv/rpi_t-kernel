1. Java Execution Environment Installation

Eclipse is built in Java, so the Java Execution Environment is needed to run Eclipse.

If the Java Execution Environment is not installed or is out of date, please install the Java Execution Environment.

Download the Java Execution Environment installer file from the following web page and install it on the development host PC.

  • Windows version Java download

The 32-bit version of the Java Execution Environment is required even when the development host PC OS is Windows 64-bit. Be sure to download and install the 32-bit version of the Java Execution Environment. (Windows Vista 64-bit is not supported.)
