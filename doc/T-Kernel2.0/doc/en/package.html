1. Package Overview

This software package includes all of the software required to build, run and debug the T-Kernel 2.0 system targeting the "T-Engine Reference Board (tef_em1d)".

  1. (1) T-Kernel 2.0 Source Code

    1. This includes the target-dependent software source code and build environment, starting with the T-Kernel 2.0 kernel, required to actually run the T-Kernel 2.0 system.

      • T-Kernel 2.0 Kernel

        This is the T-Kernel 2.0 kernel. This includes the T-Kernel/OS, T-Kernel/SM, and T-Kernel/DS. The source code is separated by directory into target-independent common code and target-dependent model-specific code to simplify porting to other targets.

      • T-Monitor

        This performs system startup and initialization and processes interrupts and exceptions. It also includes debug support functions. Basically, it is completely target-dependent.

      • Device Drivers

        These device drivers comply with the T-Kernel standard specifications for the following primary devices equipped on the target. Basically, it is completely target-dependent.

        • Clock (RTC)
        • Serial console
        • Touch panel and press button switch
        • LCD
        • System disk (micro SD use)

  2. (2) Development Environment

    1. This is the development environment required to build the T-Kernel 2.0 system. Please select one of the following three types to use.

      • GNU development environment for Eclipse (runs on Windows)
      • GNU development environment for Cygwin (runs on Windows)
      • GNU development environment for Linux (runs on Linux)

  3. (3) Emulator

    1. This is the emulator for the "T-Engine reference board (tef_em1d)" target and includes versions for both the Windows and Linux environments.

    2. Using the emulator enables development and debugging on a development PC even without the target hardware.

  4. (4) Other

    1. Includes the operation manual for the "T-Engine reference board (tef_em1d)" target, the hardware manual, and the ICE settings file.
