1. Java 実行環境のインストール

Eclipse は Java で作成されているため、実行するためには Java の実行環境が必要となります。

Java の実行環境がまだインストールされていない場合、またはバージョンが古い場合は、Java の実行環境をインストールしてください。

以下の Web ページから Java 実行環境のインストールファイルをダウンロードして、開発用ホスト PC にインストールしてください。

  • Windows版 Java のダウンロード

なお、開発用ホスト PC の OS が Windows 64 bit 版の場合でも、32 bit 用の Java 実行環境が必要になります。必ず 32 bit 用の Java 実行環境をダウンロードしてインストールしてください。( Windows Vista の 64 bit版には対応しておりません。)
