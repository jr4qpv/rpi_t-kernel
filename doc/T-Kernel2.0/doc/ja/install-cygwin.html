Cygwin のインストール

  • 1. Cygwin のインストール
  • 2. Cygwin インストール後の設定

1. Cygwin のインストール

T-Kernel 開発環境として使用する GNU 開発環境は、Windows の Cygwin 環境で動作するため、まず、Cygwin をインストールする必要があります。以下の手順で、Cygwin のインストールを行ってください。

  • すでに Cygwin がインストールされている場合も、Cygwin 1.5 以前のバージョンの場合は、正常に動作しませんので、インストールし直してください。

  • Cygwin のインストール作業は、管理者の権限のあるユーザとして、Windows にログオンして行ってください。

  1. (1) 本パッケージの "develop" フォルダにある Cygwin のインストールファイル "cygwin_d-1.7.17-1.zip" を、インストールするドライブ ("C:\temp" など) にコピーしてください。続いて、コピーしたファイルを Windows の「エクスプローラ」上でマウス右ボタンをクリックし、メニューの「すべて展開」を実行してください。

  2. (2) 展開したフォルダの中にインストーラ "setup.exe" があります。この "setup.exe" をダブルクリックしてインストーラを起動してください。

  3. (3) インストーラのウインドウが開きますので、「次へ(N)>」を押します。

  4. (4) 「Install from Local Directory」を選択して「次へ(N)>」を押します。

  5. (5) Cygwin におけるルートディレクトリを Windows 上でどこにするかを指定します。デフォルトでは "C:\cygwin" になっていますので、問題がなければデフォルトのままにしてください。

  6. (6) 「Install For」はデフォルトの「All Users」を選択してください。さらに「次へ(N)>」を押します。

  7. (7) 「Local Package Directory」は "cygwin_d-1.7.17-1.zip" を展開したフォルダを指定してください。「次へ(N)>」を押すと、ファイルのチェックが行われます。

  8. (8) しばらくすると Cygwin 1.7 系列の最初のインストール時には「Setup Alert」ダイアログが表示されます。旧バージョンの Cygwin を使用していて、それに対して上書きでインストールすると、何らかの影響が出る可能性がありますので WARNING の内容をよく読んでから作業を行ってください。

  9. (9) 次にインストールパッケージの選択画面になります。「All () Default」の「Default」をクリックし、「Install」の状態にしてから「次へ(N)>」を押すとインストールが開始されます。

    「Resolving Dependencies : W32api」ダイアログが表示された場合は、そのまま「次へ(N)>」を押してください。

  10. (10) 最後にデスクトップとスタートメニューにアイコンを作成するか聞いてきますので、そのまま「完了」を押してインストーラを終了してください。

  11. (11) インストール後に Windows のバージョンによっては「このプログラムは正しくインストールされなかった可能性があります」画面が出る場合があります。その場合、「このプログラムは正しくインストールされました」を押してください。

  12. (12) 上記のインストールでは、 GNU 開発環境を動作させるために最低限必要なもののみがインストールされます。他の機能が必要な場合は、再度、"setup.exe" を実行し、「Install from Internet」を選択して、必要な機能を追加インストールしてください。

2. Cygwin インストール後の設定

Cygwin のインストールが終わったら、インストールを行った管理者の権限のまま Cygwin 起動用のバッチファイルを修正します。

Cygwin のルートディレクトリは、Windows から見ると "C:\cygwin" になります。例えば、Cygwin 上で "/usr" と表現されるディレクトリは、Windows のエクスプローラから見た場合には "C:\cygwin\usr" のフォルダになります。

  1. (1) ファイル名の大文字と小文字を区別する設定

    Cygwin のデフォルトの状態では、ファイル名の大文字と小文字を区別しませんが、区別しないと問題を生じる場合がありますので、大文字と小文字を区別する設定に変更します。

    Cygwin の起動時に実行されるバッチファイル "C:\cygwin\Cygwin.bat" をテキストエディタで開き、

        bash --login -i

    の前に

        set CYGWIN=check_case:strict

    を追加してください。

    バッチファイルを修正した後、デスクトップ上の「Cygwin」アイコン、または、"C:\cygwin" フォルダ内の "Cygwin.bat" をダブルクリックしてください。Cygwin が起動して、Cygwin コンソールが開きます。

    Cygwin の操作は、すべて Cygwin コンソール内でコマンドシェルの bash に対してコマンドを入力して行います。

    Cygwin を終了するには、Cygwin コンソール上で "exit" を入力します。

        $ exit

インストールしたユーザと実際に使用するユーザが違う場合は、Windows から一度ログオフし、実際に使用するユーザとしてログインし直してから Cygwin を起動してください。最初の起動時にユーザの "home" ディレクトリが初期化されます。

必要に応じて、bash の起動コマンドファイル "/home/ユーザ名/.bashrc" を使いやすいように編集してください。

"C:\temp" などにコピーした "cygwin_d-1.7.17-1.zip" のファイルは、インストールの終了後は不要ですので削除してください。
