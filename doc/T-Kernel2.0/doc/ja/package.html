1. パッケージ概要

本ソフトウェアパッケージは、「T-Engine リファレンスボード (tef_em1d)」をターゲットとした T-Kernel 2.0 システムを構築(ビルド)して、実行やデバッグを行うために必要なソフトウェアをすべて含んでいます。

  1. (1) T-Kernel 2.0 ソースコード

    1. T-Kernel 2.0 のカーネル本体をはじめ、実際に T-Kernel 2.0 システムを実行するために必要となるターゲットに依存したソフトウェアのソースコードとビルド環境を含んでいます。

      • T-Kernel 2.0 カーネル

        T-Kernel 2.0 のカーネル本体です。T-Kernel/OS、T-Kernel/SM、T-Kernel/DS を含みます。これらのソースコードは、ターゲット非依存の共通部分とターゲットに依存した機種依存部分にディレクトリ単位で分離されており、別のターゲットへの移植が容易になっています。

      • T-Monitor

        システムの起動や初期化、割込みや例外などの処理を行います。また、デバッグサポート機能も含んでいます。基本的にすべてターゲット依存部分です。

      • デバイスドライバ

        ターゲットに搭載されているデバイスのうち、以下の主要デバイスの T-Kernel 標準仕様に準拠したデバイスドライバです。基本的にすべてターゲット依存部分です。

        • 時計 ( RTC )
        • シリアルコンソール
        • タッチパネル、および押しボタンスイッチ
        • LCD
        • システムディスク ( micro SD 用)

  2. (2) 開発環境

    1. T-Kernel 2.0 システムをビルドするために必要な開発環境です。以下の 3 種類を含んでおり、いずれかを選択して使用できます。

      • Eclipse 用 GNU 開発環境 ( Windows 上で動作 )
      • Cygwin 用 GNU 開発環境 ( Windows 上で動作 )
      • Linux 用 GNU 開発環境 ( Linux 上で動作 )

  3. (3) エミュレータ

    1. ターゲットである「T-Engine リファレンスボード (tef_em1d)」用のエミュレータで、Windows 環境用と Linux 環境用の両方を含んでいます。

    2. エミュレータの利用により、ターゲットのハードウェアがなくても、開発用ホスト PC のみを使ってシステムの開発やデバッグを行うことが可能となります。

  4. (4) その他

    1. ターゲットである「T-Engine リファレンスボード (tef_em1d)」の操作説明書、ハードウェアマニュアル、および ICE 用の設定ファイルを含んでいます。
