Eclipseのインストール

  • 1. Eclipse システムのインストール
  • 2. T-Kernel 開発用プラグインのインストール
  • 3. その他の設定

1. Eclipse システムのインストール

Eclipse + CDT (C/C++ Development Tooling) の環境をインストールします。

これ以下の作業は、実際に利用するユーザのアカウントで Windows にログインした状態にて行ってください。

まず、本パッケージの "develop" フォルダにある以下のファイルを、インストールするドライブの適当なフォルダ ("C:\temp" など) にコピーしてください。

    - Eclipse + CDT             "eclipse-cpp-indigo-SR2-incubation-win32.zip"
    - Eclipse日本語化プラグイン "pleiades_1.3.4.zip"
  • Eclipse のホームページには、より新しいバージョンの Eclipse がありますが、動作確認は上記のバージョンの Eclipse で行っております。このバージョンの Eclipse (Indigo SR2) をご利用ください。

次に、上記の各 zip ファイルを、上記に記載された順に展開します。エクスプローラ上で上記のファイルを選択して右クリックし、「すべて展開」を選択してください。展開先の指定は以下のようにしてください。

    - Eclipse + CDT              "C:\"
    - Eclipse日本語化プラグイン  "C:\eclipse"

"C:\eclipse" フォルダに Eclipse が展開されます。

  • パス名に空白や日本語が入る場所にはインストールしないでください。パス名に空白が存在すると ("C:\Program Files" など)、正常に動作しません。
  • 日本語環境で使用しない場合は、「Eclipse日本語化プラグイン」の展開は不要です。
  • 「Eclipse日本語化プラグイン」を展開する際、フォルダを統合するか尋ねられますが、「はい」を押してください。

Eclipse を日本語環境で使用する場合、"C:\eclipse\eclipse.ini" の最終行に以下の記述を追加してください。この記述を削除すると英語環境に戻ります。

    -javaagent:plugins/jp.sourceforge.mergedoc.pleiades/pleiades.jar

エクスプローラを使って、Eclipse をインストールした "C:\eclipse" フォルダを開きます。この中の "eclipse.exe" が Eclipse 本体ですので、このファイルのショートカットをデスクトップに作成してください。

"C:\temp" などにコピーした各 zip ファイルは、インストールの終了後は不要ですので削除してください。

2. T-Kernel 開発用プラグインのインストール

続いて、T-Kernel 開発用プラグインをインストールします。

本パッケージの "develop" フォルダにある以下のファイルを、インストールするドライブの適当なフォルダ ("C:\temp" など) にコピーしてください。

  - プラットホーム共通部分  "org.t_engine.te.1.1.2.zip"
  - tef_em1d 対応部分       "org.t_engine.tl.tef_em1d.1.1.2.zip"
  • 旧バージョンのプラグインを使用している場合は、そのプラグインを別のフォルダに移動するか、削除してください。

次に、上記の各 zip ファイルを、上記に記載された順に展開します。エクスプローラ上で上記のファイルを選択して右クリックし「すべて展開」を選択してください。展開先の指定は "C:\eclipse" としてください。

"C:\temp" などにコピーした各 zip ファイルは、インストールの終了後は不要ですので削除してください。

3. その他の設定

Windows の環境変数の HOME が設定されていると、Cygwin の環境変数の HOME との競合により、Eclipse が正常に動作しない場合がありますので、Windows の環境変数 HOME を別の変数名 (HOME_ORG など) に変更するか、または削除してください。

  • Windows の環境変数は、「コントロールパネル」→ (「システムとセキュリティ」→ )「システム」→「詳細設定」→「環境変数」で参照/編集できます。
