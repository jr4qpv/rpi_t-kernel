4. 開発方法

T-Kernel 2.0 システムの開発方法に関して説明します。本パッケージでは以下の 3 種類の開発方法に対応しています。

  1. 4-1. Eclipse を使用した開発方法 (Windows ホスト)

    1. Windows で動作する Eclipse 開発環境を使用して開発する方法を説明します。Eclipse 開発環境では、Cygwin 上の GNU 開発環境を使用します。

    2. Eclipse 開発環境の基本的な使用方法に関してましては、Eclipse (CDT プラグイン) 関連のドキュメントを参照してください。

      1. (1) Cygwin のインストール
      2. (2) Javaのインストール
      3. (3) Eclipse のインストール
      4. (4) GNU 開発環境の設定(Eclipse 用)
      5. (5) Eclipse 作業環境の構築
      6. (6) T-Kernel システムの構築(Eclipse 用)
      7. (7) 実行とデバッグ(Eclipse 用)
      8. (8) 補足事項(Eclipse 用)

  2. 4-2. Cygwin を使用した開発方法 (Windows ホスト)

    1. Windows で動作する Cygwin 上の GNU 開発環境を直接使用して開発する方法を説明します。

      1. (1) Cygwin のインストール
      2. (2) GNU 開発環境の設定(Cygwin 用)
      3. (3) T-Kernel システムの構築(Cygwin/Linux 共用)
      4. (4) 実行とデバッグ(Cygwin/Linux 共用)

  3. 4-3. Linux を使用した開発方法 (Linux ホスト)

    1. Linux 上で動作する GNU 開発環境を直接使用して開発する方法を説明します。

      1. (1) GNU 開発環境の設定(Linux 用)
      2. (2) T-Kernel システムの構築(Cygwin/Linux 共用)
      3. (3) 実行とデバッグ(Cygwin/Linux 共用)
